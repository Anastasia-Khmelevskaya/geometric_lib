# Описание функций для окружности

## 1. Принимает число r, возвращает посчитанную площадь окружности.
-**Входящие данные**-:
r - (int или float) радиус окружности.

-**Возвращаемое значение**-:
math.pi * r * r (float) - посчитанная площадь окружности.

-**Пример вызова**-:
import math
result = area(19) # Вывод: приблизительно 1134.1149479459152

## 2. Принимает число r, возвращает посчитанную длину окружности.
-**Входящие данные**-:
r (int или float) - радиус окружности.

-**Возвращаемое значение**-:
2 * math.pi * r (float) - посчитанная длина окружности (периметр окружности).

-**Пример вызова**-:
import math
result = perimeter (5) # Вернёт приблизительно 31.416... (длина окружности с радиусом 5)

# Описание функций для треугольника
## 1. Принимает числа a, b, c, возвращает площадь треугольника (формула Герона).
-**Параметры**ー：
a, b, c (int или float) : длины сторон (> 0), соблюдены неравенства треугольника.

-**Возвращаемое значение**-:
sqrt(s * (s - a) * (s - b) * (s - c)) (float), где s = (a + b + c) / 2 — посчитанная площадь треугольника.

-**Пример вызова**-:
result = area (3, 4, 5) # Вернёт 6.0

## 2. Принимает числа a, b, c, возвращает периметр треугольника.
-**Параметры**-:
a, b, c (int или float) : длины сторон.

-**Возвращаемое значение**-:
a + b + c (float) : периметр треугольника.

-**Пример вызова**-:
result = perimeter (3, 4, 5) # Вернёт 12.0

# Документация проекта geometric_lib
## 1. Общее описание решения

Проект `geometric_lib` представляет собой набор функций для вычисления
площади и периметра базовых геометрических фигур:
прямоугольника, квадрата, круга и треугольника.

Цель текущего этапа работы — дополнить библиотеку модульными тестами
(unit tests) с использованием встроенного фреймворка `unittest`,
чтобы автоматически проверять корректность вычислений на типовых и
граничных наборах данных.

## 2. Как были созданы unit-тесты
В рамках лабораторной работы №4 библиотека `geometric_lib` была дополнена
модульными тестами на основе встроенного фреймворка `unittest`.

Работа выполнялась поэтапно:
1. Был настроен Python Interpreter в CLion и создано виртуальное окружение `.venv`.
2. Для каждой геометрической фигуры были написаны отдельные файлы тестов:
    - `test_rectangle.py`
    - `test_square.py`
    - `test_circle.py`
    - `test_triangle.py`
3. Тесты оформлены в классах, наследуемых от `unittest.TestCase`.
4. Имена тестовых методов начинаются с `test_`.
5. Для проверки результатов использованы `assertEqual` и `assertAlmostEqual`,
   что позволяет корректно сравнивать как целые значения, так и вычисления с `math.pi`.
6. Для треугольника тесты учитывают реальную сигнатуру функций:
    - `area(a, b, c)` — площадь по трём сторонам (формула Герона);
    - `perimeter(a, b, c)` — сумма трёх сторон.
7. Корректность тестов подтверждена запуском:
   ```bash
   python3 -m unittest

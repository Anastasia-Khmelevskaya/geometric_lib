# Краткое описание проекта

Этот проект решает элементарные задачи из геометрии: вычисляет площадь и периметр
(длину окружности) для двух фигур — круга и квадрата.

## Решение для круга

Передаём радиус `r`. По стандартным формулам получаем:
- **Площадь**: \( S = \pi r^2 \)
- **Длина окружности**: \( P = 2\pi r \)

Обе функции возвращают числовой результат (тип `float`).

## Решение для квадрата

Передаём длину стороны `a`. Используем формулы:
- **Площадь**: \( S = a^2 \)
- **Периметр**: \( P = 4a \)

Функции выдают числовое значение результата (тип `int` или `float` — в зависимости от входных данных).

# Описание функций для окружности

## 1. Принимает число r, возвращает посчитанную площадь окружности.
-**Входящие данные**-:
r - (int или float) радиус окружности.

-**Возвращаемое значение**-:
math.pi * r * r (float) - посчитанная площадь окружности.

-**Пример вызова**-:
import math
result = area(19) # Вывод: приблизительно 1134.1149479459152

## 2. Принимает число r, возвращает посчитанную длину окружности.
-**Входящие данные**-:
r (int или float) - радиус окружности.

-**Возвращаемое значение**-:
2 * math.pi * r (float) - посчитанная длина окружности (периметр окружности).

-**Пример вызова**-:
import math
result = perimeter (5) # Вернёт приблизительно 31.416... (длина окружности с радиусом 5)

# Описание функций для окружности

## 1. Принимает число r, возвращает посчитанную площадь окружности.
-**Входящие данные**-:
r - (int или float) радиус окружности.

-**Возвращаемое значение**-:
math.pi * r * r (float) - посчитанная площадь окружности.

-**Пример вызова**-:
import math
result = area(19) # Вывод: приблизительно 1134.1149479459152

## 2. Принимает число r, возвращает посчитанную длину окружности.
-**Входящие данные**-:
r (int или float) - радиус окружности.

-**Возвращаемое значение**-:
2 * math.pi * r (float) - посчитанная длина окружности (периметр окружности).

-**Пример вызова**-:
import math
result = perimeter (5) # Вернёт приблизительно 31.416... (длина окружности с радиусом 5)

# Описание функций для треугольника
## 1. Принимает числа a, b, c, возвращает площадь треугольника (формула Герона).
-**Параметры**ー：
a, b, c (int или float) : длины сторон (> 0), соблюдены неравенства треугольника.

-**Возвращаемое значение**-:
sqrt(s * (s - a) * (s - b) * (s - c)) (float), где s = (a + b + c) / 2 — посчитанная площадь треугольника.

-**Пример вызова**-:
result = area (3, 4, 5) # Вернёт 6.0

## 2. Принимает числа a, b, c, возвращает периметр треугольника.
-**Параметры**-:
a, b, c (int или float) : длины сторон.

-**Возвращаемое значение**-:
a + b + c (float) : периметр треугольника.

-**Пример вызова**-:
result = perimeter (3, 4, 5) # Вернёт 12.0


# Описание функций для прямоугольника
## 1. Принимает числа width и height, возвращает площадь прямоугольника.

-**Параметры**ー：
width, height (int или float) : стороны прямоугольника.

-**Возвращаемое значение**-:
width * height (float) : посчитанная площадь прямоугольника.

-**Пример вызова**-:
result = area (2.5, 7) # Вернёт 17.5

## 2. Принимает числа width и height, возвращает периметр прямоугольника.
-**Параметры**-:
width, height (int или float) : стороны прямоугольника.

-**Возвращаемое значение**-:
2 * (width + height) (float) : периметр прямоугольника.

-**Пример вызова**-:
result = perimeter (3, 4) # Вернёт 14.0
